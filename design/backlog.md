# Agile Backlog (MoSCoW) - AureliusPress

## Must Have (M) - Stage 1 (MVP Deployment)
- **As an Admin**, I want to create securely log in so that I can manage content.
- **As an Admin**, I want to draft, edit, and categorize Blog Posts so that my thoughts are structured.
- **As an Admin**, I want to attach simple content / text fields so that I can add body paragraphs to my Blog Posts.
- **As a System**, I must pass all internal RSpec tests to ensure core domain logic is bug-free before pushing to production.
- **As a Developer**, I want to seed a master Admin account so I can authenticate on the live Alma VPS.

## Should Have (S) - Stage 2 (Hotwire & Aesthetics)
- **As a User**, I want to see a premium, Stoic-inspired interface (Tyrian Purple) so that the content feels authoritative.
- **As an Admin**, I want dynamic, non-reloading form saves via Hotwire (Turbo Frames/Streams).

## Could Have (C) - Stage 2
- **As an Admin**, I could integrate Editor.js or ActionText/Trix to replace basic text fields.
- **As a User**, I want instantaneous Like and Reaction mechanisms.

## Won't Have (W) - Stage 1
- Full bespoke Gutenberg-style React editor block functionality.
- Complex user subscriptions or premium memberships.
